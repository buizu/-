# モバレ プランナー

これは「モバイルレジェンド：Bang Bang」の戦術計画ツールです。  
マップ上に役割（E, R, J, M, G）を配置したり、線を引いたりすることで、戦略を視覚的に共有できます。

![サンプル画面](images/mbrmap.jpg)

---

## 🔧 機能

- ロール（役割）の配置とドラッグ移動
- 線の描画（複数色対応）
- 消しゴムで線を削除
- 初期位置へのリセット
- モバイル対応

---

## 🖱️ 操作方法

| モード | 説明 |
|--------|------|
| ✋ ドラッグモード | ロールを動かす |
| ✏️ ペイントモード | 線を引く（色選択可） |
| 🩹 消しゴムモード | 線を消す |
| 🔄 リセット | 初期状態に戻す（確認あり） |
